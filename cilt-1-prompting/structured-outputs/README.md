# Structured Outputs

**Cilt:** 1 — Prompt Mühendisliği ve Structured Output

JSON schema, Pydantic, Zod ve doğrulama örnekleri.

## JSON Repair Standardı

1. Önce schema ile doğru üretmeye zorla.
2. Çıktıyı doğrula.
3. Kritik olmayan akışta repair dene.
4. Kritik akışta retry, fallback veya insan onayı uygula.
