Açıkla, hipotez kur, kanıt göster

1. Açıkla

Bozuk fonksiyon, assigned=[] listesini fonksiyon tanımlanırken bir kez oluşturur. Parametre verilmeden yapılan sonraki çağrılar aynı listeyi kullanır.

2. Hipotez kur

İkinci planın A vardiyasını taşımasının nedeni plan nesnesi değildir; iki çağrı arasında paylaşılan değişebilir varsayılan listedir.

3. Kanıt göster

Minimal repro, iki planın vardiya listelerini kimlik karşılaştırmasıyla denetler. Bozuk sürümde birinci.shifts === ikinci.shifts sonucu doğrudur.

def add_shift_to_schedule(schedule, shift, assigned=None):
    if assigned is None:
        assigned = []
    assigned.append(shift)
    schedule.shifts = assigned
    return schedule

assigned = assigned or [] kullanılmaz. Çağıran taraf bilerek boş bir liste verdiğinde bu liste korunmalıdır.
