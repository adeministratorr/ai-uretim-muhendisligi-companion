# kod/ — Kitap içi kalıcı kod bağlantıları

Kitaba gömülen "kodu okuyun" bağlantıları `lab.ademyuce.tr/kod/<yol>/`
biçimindedir ve buradaki sayfalar üzerinden GitHub depo görünümüne yönlenir.

KURAL: Kitap yayımlandıktan sonra buradaki hiçbir yol silinemez veya yeniden
adlandırılamaz (bkz. kitap deposundaki `yayin_permalink_semasi.md`). Depo
taşınırsa yalnız bu sayfaların hedefi güncellenir; kitaptaki bağlantılar
kırılmaz.
